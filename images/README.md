# Dossier des photos — La Réserve

Ce dossier accueille les photos du restaurant. Tant qu'il est vide,
le site affiche de jolis blocs gris de remplacement : **le site reste
présentable même sans aucune photo.** Il devient magnifique une fois
les vraies photos ajoutées.

## Comment ajouter une photo

1. Déposez votre image (format `.jpg`) dans ce dossier `images/`.
2. Ouvrez `index.html` et cherchez le commentaire `PHOTO À VENIR`
   correspondant.
3. Remplacez le bloc `<div class="placeholder">…</div>` par la balise
   `<img …>` indiquée juste au-dessus dans le commentaire.

> Astuce : gardez exactement les noms de fichiers proposés ci-dessous,
> comme ça vous n'avez presque rien à modifier dans le code.

## Photos attendues et dimensions conseillées

| Fichier            | Emplacement sur le site      | Format   | Taille minimale | Ce qu'il faut photographier                          |
| ------------------ | ---------------------------- | -------- | --------------- | ---------------------------------------------------- |
| `hero-four.jpg`    | Grande image d'accueil       | Paysage  | 2000 × 1200 px  | Le four à bois en pleine flambée, cadrage large sombre |
| `feu.jpg`          | Section « Le feu de bois »   | Carré    | 800 × 800 px    | Gros plan sur les flammes ou une pizza au sortir du four |
| `pizza.jpg`        | Catégorie Pizzas             | Carré    | 600 × 600 px    | Une pizza vue du dessus (la Paysanne par exemple)    |
| `salade.jpg`       | Catégorie Salades            | Carré    | 600 × 600 px    | Une salade italienne vue du dessus                   |
| `viande.jpg`       | Catégorie Viandes            | Carré    | 600 × 600 px    | Une entrecôte grillée au feu de bois                 |
| `dessert.jpg`      | Catégorie Desserts           | Carré    | 600 × 600 px    | Un tiramisu maison                                   |
| `og-four.jpg`      | Aperçu réseaux sociaux       | Paysage  | 1200 × 630 px   | Une belle photo du four (partage Facebook/WhatsApp)  |

## Conseils pour de belles photos

- **Lumière naturelle** de préférence, ou la lueur chaude du four.
- Les photos de plats sont plus appétissantes **vues du dessus**.
- Évitez les images trop lourdes : visez moins de **500 Ko** par photo
  (vous pouvez les compresser sur un site comme squoosh.app).
- Cadrez **carré** pour les plats : ils s'afficheront en rond sur le site.
